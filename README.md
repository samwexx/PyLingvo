# PyLingvo

PyLingvo is a 2D educational game developed using Unity.
The project was created as a study project.

## About the Project
PyLingvo is a prototype game focused on basic gameplay mechanics,
player interaction, and NPC behaviour.
The main goal of the project is to explore game development fundamentals.

## How to Run
There are two ways to use the project:

- **Unity project**:  
  Open the project via Unity Hub.

- **Build version**:  
  Download the Windows build and run the `.exe` file.

## Technologies Used
- Unity
- C#

## Features
- Player movement
- NPC interaction
- Task-based gameplay
